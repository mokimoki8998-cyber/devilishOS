# 😈 devilishOS (v0.15) — Stable Custom Console & Shell

![devilishOS Logo](watermarked_img.png)

**devilishOS** — это легковесная 16-битная операционная система (графическая оболочка) для DOS, написанная на языке ассемблера x86. Проект представляет собой независимую среду со своей системой авторизации, графическим рабочим столом, поддержкой мыши и встроенным командным интерпретатором.

---

## 🚀 Возможности системы

### 🔒 Безопасность и пользователи
Система имеет полноценный экран авторизации с интерактивным текстовым меню:
* **Вход (Login):** Авторизация по логину и маскируемому паролю с проверкой по локальной базе данных `USERS.DAT`.
* **Регистрация (Register):** Быстрое добавление нового пользователя в систему.
* **Сброс (Factory Reset):** Полная очистка базы пользователей (требуется ввод мастер-пароля `RES`).

### 🖥 Графический рабочий стол (GUI)
После успешного входа инициализируется графический режим VGA (640x480, 16 цветов):
* **Поддержка мыши:** Полноценный курсор (`int 33h`) для взаимодействия с элементами интерфейса.
* **Динамический рабочий стол:** Автоматический сканер текущего диска на наличие исполняемых файлов `*.COM` (исключая системные файлы) с выводом их иконок на рабочий стол.
* **Быстрый запуск:** Возможность запустить любую найденную программу прямо кликом по иконке.
* **Меню [ START ]:** Меню со встроенной системной информацией, консолью и блокировкой ОС.

### 📟 Консоль (Devilish Console)
Окно консоли выполнено в классическом оконном интерфейсе с кнопкой закрытия `[X]` и поддерживает набор команд:
* `dir` — список доступных файлов.
* `cls` — очистка экрана терминала.
* `f:` — имитация переключения на диск F:.
* `666` — вызов критического сбоя системы (BSOD) с последующей перезагрузкой по клавише `ESC`.
* Прямой запуск внешних `.COM` файлов через ввод их имени.

## 🛠 Технический стек
* **Архитектура:** 16-битный реальный режим процессора (Real Mode).
* **Формат:** Плоский бинарный файл `.COM` (`org 100h`).
* **Прерывания:**
  * `int 10h` — вывод графики, работа с видеорежимами.
  * `int 16h` — ввод с клавиатуры.
  * `int 21h` — работа с файлами и запуск сторонних процессов (функция `4B00h`).
  * `int 33h` — работа с драйвером мыши.

---

## 📂 Структура файлов
* `main.asm` — исходный код операционной системы.
* `USERS.DAT` — база данных зарегистрированных пользователей.
